**Project Requirements Summary for Industrial Sensor Data Platform**

The **Industrial Sensor Data Platform** is a unified solution for collecting, processing, and analyzing sensor data from diverse sources—including SCADA systems, mainframes, and IoT devices—to enable real-time monitoring, predictive maintenance, and operational optimization. Core functionalities encompass **data ingestion** (via APIs/protocols like OPC-UA), **real-time and batch processing**, **customizable reporting**, and **automated email alerts**, all accessible through an intuitive web dashboard. The platform supports **interactive mode** for live monitoring, **batch mode** for scheduled analytics, and **legacy system integration** (e.g., mainframe imports), while a **sensor monitoring daemon** ensures continuous uptime. Designed for both technical and non-technical stakeholders, the platform bridges gaps between engineering teams and business leaders by providing actionable insights without requiring deep domain expertise.
