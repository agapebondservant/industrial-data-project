The **Industrial Data Platform** is designed to collect, process, and analyze sensor data from industrial environments, providing real-time reporting, automated email alerts, and an interactive web dashboard for monitoring. It seamlessly integrates with **SCADA systems, mainframes, and other data sources**, ensuring compatibility with diverse industrial infrastructure. The platform supports multiple usage modes, including **interactive operations, batch processing for scheduled tasks, mainframe data imports, and dedicated sensor monitoring via serial/MQTT interfaces**, making it adaptable to various industrial workflows.
