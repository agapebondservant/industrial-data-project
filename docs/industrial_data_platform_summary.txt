The **Industrial Data Platform** is designed to collect, process, and unify industrial sensor data from SCADA systems, mainframes, and other sources for real-time reporting, email alerts, and dashboard visualization. Key features include interactive and batch data processing modes, a sensor monitoring daemon for MQTT-based sensor data, and integration with legacy systems like mainframes. Dependencies include external systems (SCADA, mainframes, MQTT brokers) and internal scripts (`run_platform.py`, `batch_import.py`, `sensor_monitor.py`). This platform uniquely consolidates disparate data sources to enable efficient industrial monitoring and decision-making, while legacy features like mainframe imports remain supported.
